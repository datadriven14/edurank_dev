🔥 Konfigurasi Firebase Realtime
Untuk sinkronisasi realtime antar semua komputer
Terkonfigurasi ✓
📋 Cara Setup Firebase (sekali saja, ~10 menit):
1.Buka console.firebase.google.com → Add Project → beri nama (mis: "edurank-ai")
2.Di sidebar → Realtime Database → Create Database → pilih region → Start in test mode
3.Di sidebar → Project Settings (⚙️) → Your apps → Web (</>) → Register app
4.Copy nilai firebaseConfig yang muncul (apiKey, databaseURL, dll)
5.Paste ke form di bawah → klik Simpan & Hubungkan
⚠️Test mode aktif 30 hari. Setelah itu update Rules di Firebase Console.
🌐 Cloudflare Worker — Realtime Tanpa Config
Semua komputer terhubung otomatis, tidak perlu isi apapun di komputer lain
Terkonfigurasi ✓
📋 Cara Setup (~15 menit, sekali saja):
1.Daftar gratis di cloudflare.com → Workers & Pages → Create → Create Worker
2.Beri nama edurank-ai → Edit Code → paste isi cloudflare-worker.js → Save & Deploy
3.Workers & Pages → KV → Create namespace → nama: EDURANK_KV
4.Worker → Settings → Variables → KV Namespace Bindings → Add → Variable: KV, Namespace: EDURANK_KV → Save
5.Copy URL Worker → paste di bawah → klik Simpan & Aktifkan
6.Upload index.html baru ke GitHub → selesai!
✓Komputer lain tidak perlu setting apapun — buka URL langsung terhubung
⚠️ Setelah Simpan, upload ulang index.html ke GitHub agar URL tertanam di kode dan komputer lain tidak perlu setting.
⚙️ Konfigurasi Google Apps Script
URL harus diakhiri /exec — bukan /dev
📋 Cara Setup (ikuti urutan ini):
1. Buka Google Sheets baru → Extensions → Apps Script
2. Hapus semua kode default, paste isi file gas-backend.gs
3. Klik Run → setup() — izinkan akses saat diminta
4. Deploy → New Deployment → Web App
Execute as: Me  |  Who has access: Anyone
5. Copy URL deployment (akhiran /exec) → paste di atas
6. Klik 🔌 Test dulu — pastikan muncul ✅ sebelum sync
⚠️ Setiap edit kode GAS → wajib Deploy ulang (bukan pakai deployment lama)
